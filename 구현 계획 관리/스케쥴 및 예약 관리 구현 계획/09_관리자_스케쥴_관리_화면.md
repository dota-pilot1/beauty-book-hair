# 09. 관리자 스케쥴 관리 화면

## 목표

관리자가 예약 가능한 시간의 기준을 직접 관리할 수 있게 한다.

## 페이지

```text
beauty-book--front/src/app/schedules/page.tsx
```

또는 관리 메뉴 기준으로:

```text
beauty-book--front/src/app/admin/schedules/page.tsx
```

현재 라우팅 구조에 맞춰 선택한다.

## 화면 구성

### 상단

- 제목: 스케쥴 관리
- 설명: 영업시간, 디자이너 근무시간, 예약 불가 시간을 관리합니다.

### 탭

1. 영업시간
2. 디자이너 근무시간
3. 예약 불가 시간
4. 디자이너 가능 시술

## 영업시간 탭

요일별 카드 또는 테이블.

- 요일
- 영업 여부
- 오픈 시간
- 마감 시간

## 디자이너 근무시간 탭

- 디자이너 선택
- 요일별 근무 여부
- 시작 시간
- 종료 시간

## 예약 불가 시간 탭

- 날짜 선택
- 디자이너 선택
- 시작 시간
- 종료 시간
- 사유
- 목록
- 추가/삭제

## 디자이너 가능 시술 탭

- 디자이너 선택
- 전체 시술 목록
- 수행 가능 여부 체크
- 저장

이 설정은 고객 예약 화면의 디자이너 후보 목록과 예약 생성 검증에 사용된다.

## UI 원칙

- 캘린더 드래그 UI는 1차에서 하지 않는다.
- 폼과 목록 중심으로 빠르게 검증한다.
- 시간 입력은 select 또는 time input을 사용한다.

## 완료 기준

- 영업시간을 수정하면 고객 예약 슬롯이 바뀐다.
- 디자이너 근무시간을 수정하면 해당 디자이너 슬롯이 바뀐다.
- 예약 불가 시간을 추가하면 해당 시간대 슬롯이 BLOCKED 상태로 보인다.
- 디자이너 가능 시술을 수정하면 고객 디자이너 선택 목록이 바뀐다.
