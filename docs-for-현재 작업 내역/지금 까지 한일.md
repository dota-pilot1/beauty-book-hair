## 게시판 시스템 전체 구현 (이번 세션)

### 백엔드
- `Board` 엔티티/서비스/컨트롤러 신규 구현 (DDD 레이어 구조)
- `BoardConfig` (게시판 설정), `Board` (게시글), `BoardComment` (댓글) 엔티티
- `@SQLRestriction("deleted_at IS NULL")` 소프트 삭제
- 공개 API: `GET /api/boards/configs`, `GET /api/boards/{code}`, `GET /api/boards/{code}/{id}`
- 관리자 API: 게시판 설정 CRUD, 게시글 CRUD, 고정(핀) 토글
- 댓글 API: `GET/POST /api/boards/{boardId}/comments`, `DELETE /api/boards/{boardId}/comments/{commentId}`
- SecurityConfig — 공개 게시판 GET 엔드포인트 `permitAll` 추가
- **메뉴 자동 연동**: 게시판 생성 시 헤더 1차 메뉴 `BOARD` (없으면 자동 생성) + 2차 메뉴 `BOARD_<CODE>` 자동 추가; 삭제 시 메뉴도 함께 삭제

### 프론트엔드
- `src/entities/board/` — `types.ts`, `boardApi.ts`, `useBoards.ts` (TanStack Query hooks 전체)
- 게시판 관리 어드민 페이지 (`/board-management`) — 게시판 설정 CRUD
- 게시글 목록 공개 페이지 (`/boards/[code]`) — 스플릿 뷰 레이아웃
  - **왼쪽**: 번호/제목/날짜 컬럼 헤더, 공지 뱃지, 선택 인디케이터
  - **오른쪽**: 본문 상세 + 인라인 수정/삭제 + 댓글 섹션
  - 댓글: 아바타(이니셜), 관리자 뱃지, hover 삭제 버튼
  - 글쓰기 폼: split-view 밖 전체 너비 오버레이

### Lexical 리치텍스트 에디터 적용
- `lexical v0.43.0` 패키지 설치 (react, rich-text, list, code, link 등)
- `src/shared/ui/lexical/` 에디터 컴포넌트 신규 구현
  - `LexicalEditor`: 편집/읽기 모드 통합 (readOnly prop)
  - `LexicalToolbar`: Bold/Italic/Heading/List/Code/Image 툴바
  - `ImageNode` + `ImagePlugin`: 이미지 삽입·리사이즈·정렬 지원
  - `DragDropImagePlugin`: 드래그앤드롭·붙여넣기 이미지 업로드
  - `/api/upload/presign` 연동 업로드 유틸
- 글쓰기/수정 폼 및 본문 뷰에 에디터 적용
- `InitialContentPlugin` `flushSync` 에러 수정 (`queueMicrotask` 사용)

### UI 엔터프라이즈 리디자인
