## 2. 지금까지 한 작업

### Designer → Staff 전면 리네임
- `Designer` 엔티티를 `Staff`로 교체하고 `StaffRole` enum(DESIGNER / STAFF / DESK) 추가
- `DesignerWorkingHour` → `StaffWorkingHour`, `BlockedTime` FK `designer_id` → `staff_id`
- `Reservation` FK `designer_id` → `staff_id`
- API URL: `/api/designers` → `/api/staff`, `/api/admin/designers` → `/api/admin/staff`
- `DesignerWorkingHourRepository` → `StaffWorkingHourRepository`
- ErrorCode: `DESIGNER_NOT_FOUND` → `STAFF_NOT_FOUND` 등
- 프론트 타입 `AvailableDesigner` → `AvailableStaff`, 필드 `availableDesigners` → `availableStaff`

### 직원 관리 화면 신규 구현
- 관리 메뉴에 "직원 관리" 항목 추가 (MenuSeeder)
- `/staff` 페이지: 테이블 + 우측 상세 패널 (5:5 분할)
- 우측 패널: 프로필 / 소개 / 가능 시술 토글

### 버그 수정
- `ScheduleReservationSeeder`: 서버 재시작 시 Staff `profileImageUrl` 덮어쓰는 버그 수정
- `BeautyService.replaceImageUrls`: 같은 리스트 참조로 인해 이미지가 clear되는 버그 수정
