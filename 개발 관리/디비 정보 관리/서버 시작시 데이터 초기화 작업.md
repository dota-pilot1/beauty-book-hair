# 서버 시작 시 데이터 초기화 작업

Spring Boot `ApplicationRunner` + `@Order` 패턴으로 서버 구동 시 자동 시드 데이터를 삽입합니다.
모든 Seeder는 **멱등성 보장** — `existsByCode()` 또는 `findByCode()` 로 중복 삽입을 방지합니다.

---

## 실행 순서 (`@Order`)

| Order | 클래스 | 설명 |
|------:|--------|------|
| 1 | `PermissionCategorySeeder` | 권한 카테고리 (USER, ROLE, PERMISSION …) |
| 2 | `RoleSeeder` | 역할 (ROLE_USER, ROLE_MANAGER, ROLE_ADMIN) |
| 3 | `PermissionSeeder` | 세부 권한 (VIEW/EDIT/DELETE) |
| 4 | `MenuSeeder` | 내비게이션 메뉴 트리 |
| 5 | `TestCustomerUserSeeder` | 테스트 고객 계정 |
| 6 | `BeautyServiceCategorySeeder` | 시술 카테고리 5종 |
| 7 | `BeautyServiceCategoryBackfillRunner` | 레거시 카테고리 마이그레이션 |
| 8 | `BeautyServiceSeeder` | **시술 항목 세부화 (27종)** |
| 9 | `ScheduleReservationSeeder` | 직원·근무시간·영업시간·차단시간 |

---

## 시술 카테고리 (`BeautyServiceCategorySeeder`, Order 6)

| code | 이름 | 설명 |
|------|------|------|
| CUT | 컷 | 기본 컷, 레이어드 컷 등 컷 시술 |
| PERM | 펌 | 볼륨펌, 셋팅펌 등 펌 시술 |
| COLOR | 염색 | 뿌리염색, 전체염색 등 컬러 시술 |
| CLINIC | 클리닉 | 케어 및 손상 회복 중심 시술 |
| STYLING | 스타일링 | 드라이, 스타일링, 업스타일 |

## 직원 & 스케줄 (`ScheduleReservationSeeder`, Order 9)

### 영업시간
- 월~토 10:00 ~ 20:00 / 일요일 휴무

### 디자이너

| 이름 | 소개 | 근무시간 |
|------|------|---------|
| 하린 | 레이어드 컷과 컬러 상담을 함께 제안합니다. | 월~토 10:00~19:00 |
| 민서 | 손상도에 맞춘 클리닉과 데일리 스타일링을 담당합니다. | 월~토 10:00~19:00 |

### 직원-시술 매핑
- **하린**: 전체 시술 가능 (active=true)
- **민서**: 처음 2개 시술만 active=true (나머지는 등록되나 비활성)

### 차단시간 (BlockedTime)
- 하린 / 매주 월요일 13:00~14:00 / 점심 (개발용 샘플)
